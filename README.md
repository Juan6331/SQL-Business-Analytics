# SQL-Business-Analytics
Proyecto SQL para analizar datos de ventas minoristas con el fin de extraer información útil para el negocio mediante agregaciones, CTE y funciones de ventana

- Consultas SQL (SELECT, WHERE, ORDER BY)
- Agregaciones (SUM, AVG, COUNT)
- Operaciones JOIN
- Expresiones de tabla común (CTE)
- Funciones de ventana (RANK, ROW_NUMBER)
